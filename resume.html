resume belajar git

  1. membuat file html
  2. buka terminal di vscode
  3. melakukan configurasi untuk yang baru pertama kali menginstal git
  4. setelah mengisi file html, melakukan inisiasi di terminal dengan cara mengetikkan "git init"
  5. setelah itu, mengetikkan "git add (nama file)" atau "git add ." untuk menambahkan perubahan pada file yang telah di edit
  6. setelah itu, mengetikkan "git commit -m "deskripsi perubahan" untuk menyimpan perubahan
  7. membuat repository di github
  8. membuat branch master dengan cara mengetikkan "git checkout -b (nama branch)"
  9. menambahkan repository yang sudah di buat dengan cara "git add remote origin master (link repository)"
  10. setelah menambahkan remote repository, kita melakukan push dengan cara mengetikkan pada terminal "git push --set-upstream origin Master"
  11. untuk mengambil codingan dari repository orang lain mengetikkan "git clone (link repository yang akan di clone)"
  12. untuk mengupdate hasil cloningan, ketikkan "git pull"
  13. untuk switch branch, ketikkan pada terminal "git checkout (nama branch)"
